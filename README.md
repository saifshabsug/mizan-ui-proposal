# ميزان — بروبوزال تصميم الواجهة · Mizan UI/UX Design Proposal

صفحة مقترح تصميم تفاعلية لإعادة تصميم واجهة نظام ميزان المحاسبي.
ملف واحد مستقل بالكامل — فاتح/غامق، عربي RTL / إنجليزي، خط Cairo مضمّن، بدون أي اعتماد خارجي.

An interactive UI/UX design proposal for the Mizan accounting system — a single, fully self-contained page (light/dark, Arabic-RTL / English, embedded Cairo font, no external dependencies).

## معاينة محلياً · View locally
افتح `index.html` في أي متصفّح.

## النشر على Vercel · Deploy to Vercel
1. افتح **https://vercel.com/new**
2. استورد هذا المستودع (`mizan-ui-proposal`).
3. Framework Preset: **Other** — بدون Build Command — الإخراج من الجذر.
4. اضغط **Deploy** → بيطلع رابط `*.vercel.app` عام تشاركه.
